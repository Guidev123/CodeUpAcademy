        <h1 style="color: #0056b3; text-align: center;">🚀 CodeUpAcademy</h1>
        <p><strong>CodeUpAcademy</strong> is an e-learning platform designed for programming education. The system follows a modular monolithic architecture with a strong emphasis on software architecture best practices. To access the courses, users must subscribe to a paid plan. 🎓</p>
      
  ![modular](https://github.com/user-attachments/assets/fc2c4dc6-f6b1-406b-83ee-0e2ac56fba0b)      
      
        
   <h2 style="color: #0056b3;">🛠️ Architecture</h2>
        <ul>
            <li>🧩 Modular Monolith</li>
            <li>🔌 <strong>Ports and Adapters</strong> Architecture</li>
            <li>🧑‍💻 Principles of <strong>Clean Architecture</strong> and <strong>Domain Driven Design</strong></li>
        </ul>
        
  <h2 style="color: #0056b3;">📌 Modules</h2>
        <ul>
            <li><strong>🔐 Authentication</strong> - User management and authentication.</li>
            <li><strong>🎓 Students</strong> - Student control and course progress tracking.</li>
            <li><strong>💳 Subscriptions</strong> - Subscription and payment management.</li>
            <li><strong>📚 Courses</strong> - Course and content administration.</li>
        </ul>
        
   <h2 style="color: #0056b3;">🔄 Communication</h2>
        <p>The modules communicate asynchronously via the <strong>Event Bus</strong> using <strong>RabbitMQ</strong>. 📡</p>
        
   <h2 style="color: #0056b3;">🚧 Project Status</h2>
        <p style="color: red;"><strong>Incomplete</strong> - Still under development. ⏳</p>
